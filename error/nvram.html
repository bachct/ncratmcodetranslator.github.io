NVRam Clear:

22e DISPENSER NVRAM CLEAR

  1. 1. Power off ATM 2. Pull out dispenser and remove battery from board. Unclips from R/H side. Replace after 60 seconds. 3. Rack dispenser back into position and power on ATM with NCR usb key inserted. 4. When booted up, enter into diagnostics/sys app/ and select number 8 self service configuration 5. Select 1 set device config/ then 1 usb cash dispenser/ then 1 note configuration and the dispenser will run a clear 6. Select 1 set 7. Display will show note size type 1 through to 4 as 66 8. Tab down to singularity type 1 and enter in 102 9. Tab down to singularity type 2 and enter 107 10. 3 and 4 can remain as 255 so tab down to page down and press page down on right hand side fdk menu. 11. Leave presentation order as 1234 and press ok 12. Display will show configure new parameters, toggle to yes and press ok 13. Hit previous and select 2 device parameter configuration 14. Dispense protection configuration should be USB. If logical or physical, toggle to usb and press ok. 15. Cancel back out to the main system application menu and check 5 device status list. Complete a clear all if required and exit back to main menu to complete a test cash. The ATM may reset itself when exiting to the main menu. If so wait for full reboot and complete test cash and bring the ATM back into service and complete a test transaction. 5877, 5870 Bunch and 5884 Dispenser.

P4 & P3 SDC NVRam Clear:


The bank of 8 switches on the SDC board.

    Switches 3 and 8 on.
    Power the ATM up.
    Give it 45 seconds.
    Turn the ATM power off.
    Reset the 2 dip switches.
22e SDC NVRam Clear:

    Remove the onboard battery.
    Give it 45 seconds.
    Replace the battery.
