ATM Error Code Translator

NVRam Clear:

22e DISPENSER.
  1. Power off ATM.
  2. Pull out dispenser and remove battery from board. Unclips from R/H side. Replace after 60 seconds.
  3. Rack dispenser back into position and power on ATM with NCR usb key inserted.
  4. When booted up, enter into diagnostics/sys app/ and select number 8 self service configuration.
  5. Select 1 set device config/ then 1 usb cash dispenser/ then 1 note configuration and the dispenser will run a clear.
  6. Select 1 set.
  7. Display will show note size type 1 through to 4 as 66.
  8. Tab down to singularity type 1 and enter in 102.
  9. Tab down to singularity type 2 and enter 107.
  10. 3 and 4 can remain as 255 so tab down to page down and press page down on right hand side fdk menu.
  11. Leave presentation order as 1234 and press ok.
  12. Display will show configure new parameters, toggle to yes and press ok.
  13. Hit previous and select 2 device parameter configuration.
  14. Dispense protection configuration should be USB. If logical or physical, toggle to usb and press ok.
  15. Cancel back out to the main system application menu and check 5 device status list. Complete a clear all if required and exit back to main menu to complete a test cash. The ATM may reset itself when exiting to the main menu. If so wait for full reboot and complete test cash and bring the ATM back into service and complete a test transaction.



5877, 5870, 5884 DISPENSERS.

The top bank of 8 switches.
  1. Switches 3 and 8 on.
  2. Power the ATM up.
  3. The LED's on the control board start pulsing.
  4. When they stop pulsing turn the ATM power off.
  5. Reset the 2 dip switches.
  6. The note opacities and singularities will need setting up.
  7. $20 Type1 = 102 and Type2 $50 = 107 & sizes at 66.



P4 & P3 SDC NVRam Clear:

The bank of 8 switches on the SDC board.
  1. Switches 3 and 8 on.
  2. Power the ATM up.
  3. Give it 45 seconds.
  4. Turn the ATM power off.
  5. Reset the 2 dip switches.



22e SDC NVRam Clear:

  1. Remove the onboard battery.
  2. Give it 45 seconds.
  3. Replace the battery.



CARD READER
MCRW/IMCRW
NVRAM Clear:

  1. On the main board there is a block of 4 switches, they are in front, close to the face.
  2. To reset NVRAM, you need to switch the switch 3 to ON.
  3. Turn on the power for 20-30 seconds.
  4. Turn the power off.
  5. Return switch 3 back to OFF.

  6. For the NCR 5870 reader you need to switch both 3 and 8.



NCR PRINTERS
NVRAM Clear:

Printers connected via SDC bus.
  1. Record / memorize the current position set DIP DSW2 switches.
  2. Set DIP switches 3,8 to ON.
  3. Press the RESET (SW2 button). While LEDs LEDs flash, installs the factory default settings.
  4. Return to the original position of the DIP switches.
Printers connected via COM-port.
  1. Record / memorize the current position set DIP DSW2 switches.
  2. Set DIP switches 1, 2 and 5 to ON.
  3. Press the RESET (SW2 button). While LEDs LEDs flash, installs the factory default settings.
  4. Return to the original position of the DIP switches.

IMCRW Info.

M-DATA Meanings:

  • Byte 0 = Command code
  • Byte 1 = Mag track errors: Read error, Write error or Smart commands: sub function code
  • Byte 2 = Shutter jams, card jams, HiCo write, invalid cards or Battery status
  • Byte 3 = Sensors, HiCo capability
  • Byte 4 = ICC block information or CIM overall status
  • Byte 5 = 0 or CIM detailed status
  • Byte 6 = Sankyo library function code
  • Byte 7 = 0 or Sankyo library error code.
